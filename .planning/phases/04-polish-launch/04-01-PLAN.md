  <action>
1. Install the motion library: `npm install motion`

2. Create `src/lib/motion.ts` - shared animation variants and config:
   - `sectionVariants`: hidden (opacity: 0, y: 20) / visible (opacity: 1, y: 0)
   - `staggerContainerVariants`: hidden {} / visible with staggerChildren: 0.07
   - `staggerItemVariants`: hidden (opacity: 0, y: 15) / visible (opacity: 1, y: 0, duration: 0.35)
   - `heroStaggerVariants`: same as stagger but with staggerChildren: 0.15 and item duration: 0.5
   - `defaultTransition`: { duration: 0.4, ease: "easeOut" }
   - `defaultViewport`: { once: true, margin: "-100px" as const }

3. Create `src/components/motion/scroll-reveal.tsx` - "use client" component:
   - Props: children, className (optional), delay (optional number, default 0)
   - Uses motion.div with initial="hidden", whileInView="visible", viewport={{ once: true, margin: "-100px" }}
   - Imports sectionVariants and defaultTransition from lib/motion
   - If delay is provided, adds it to the transition

4. Create `src/components/motion/stagger-children.tsx` - "use client" components:
   - `StaggerContainer`: motion.div with staggerContainerVariants, initial="hidden", whileInView="visible", viewport={{ once: true }}
   - `StaggerItem`: motion.div with staggerItemVariants (no initial/animate - inherits from parent)
   - Both accept children and optional className

5. Create `src/app/(public)/template.tsx` - page transition wrapper:
   - "use client" component, default export
   - Wraps children in motion.div with initial={{ opacity: 0 }}, animate={{ opacity: 1 }}, transition={{ duration: 0.3, ease: "easeInOut" }}
   - template.tsx remounts on every navigation, giving entry fade animation

6. Update `src/components/layout/public-navbar.tsx` - add scroll-aware hide/show:
   - Import motion, useScroll, useMotionValueEvent from "motion/react"
   - Import useState from "react"
   - Add hidden state (default false)
   - Use useScroll() to get scrollY
   - Use useMotionValueEvent(scrollY, "change", callback) to compare current vs previous scroll position
   - When scrolling down (latest > previous) and past 150px threshold, set hidden=true
   - When scrolling up, set hidden=false
   - Replace <header> with <motion.header animate={{ y: hidden ? "-100%" : "0%" }} transition={{ duration: 0.3 }}>
   - Keep all existing classes and content unchanged
  </action>
